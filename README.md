✈️ TravelAr
Simulador Interactivo de Reservas de Viajes

TravelAr es un simulador interactivo de reservas turísticas desarrollado como proyecto frontend, enfocado en experiencia de usuario, diseño moderno y simulación de lógica de negocio aplicada al rubro de viajes.

El sistema permite visualizar destinos, seleccionar paquetes y simular el proceso de reserva con cálculo dinámico de precios.
